# Breast Cancer Classification using Machine Learning

## Project Overview
This project is a Binary Classification Machine Learning system developed as part of Internship Task-4. The objective of the project is to classify breast cancer tumors as Malignant or Benign using different classification algorithms and evaluation techniques.

The project demonstrates:
- Classification model training
- Feature scaling
- Model evaluation
- Confusion matrix analysis
- Precision, Recall, and F1-score interpretation
- ROC Curve and AUC Score
- Handling class imbalance
- Hyperparameter tuning using GridSearchCV

------------------------------------------------------------

## Dataset Used
Breast Cancer Wisconsin Dataset
(Source: scikit-learn built-in dataset)

Target Classes:
0 -> Malignant
1 -> Benign

------------------------------------------------------------

## Project Workflow
1. Import required libraries
2. Load Breast Cancer dataset
3. Split dataset into training and testing sets
4. Perform feature scaling
5. Train Logistic Regression model
6. Evaluate model performance
7. Generate confusion matrix and classification report
8. Plot ROC Curve and calculate AUC Score
9. Handle class imbalance using class_weight
10. Compare with Decision Tree Classifier
11. Tune hyperparameters using GridSearchCV

------------------------------------------------------------

## Results
- Logistic Regression achieved high classification accuracy.
- ROC-AUC score showed strong predictive capability.
- Balanced Logistic Regression improved handling of class imbalance.
- Decision Tree provided competitive performance.
- GridSearchCV helped identify optimal model parameters.
